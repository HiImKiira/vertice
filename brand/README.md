# Vértice — Guía de marca

## Concepto

El símbolo representa un **vértice facetado**, dos caras inclinadas que se encuentran en un punto. Lee como la letra **V** y, al mismo tiempo, como un corte diamante de joyería fina: una cumbre. Asociaciones: precisión, cumbre, decisión, valor.

## Paleta

| Rol | Hex | Uso |
|-----|------|-----|
| Onyx | `#0A0E1A` | Texto principal sobre fondo claro, fondos premium |
| Crema | `#F4F0E8` | Fondo claro de marca, texto sobre onyx |
| Oro claro | `#F1CB7E` | Highlight superior del símbolo |
| Oro medio | `#C9A961` | Color principal del oro |
| Oro profundo | `#85692A` | Sombra inferior del oro |
| Oro tostado | `#3F2F11` | Sombra extrema en la cara derecha |
| Gris polvo | `#6B5F4C` | Texto de tagline sobre claro |

## Tipografía

- **Wordmark**: serif clásica (Georgia / EB Garamond). Peso 400, tracking +6%.
- **Tagline**: sans neutra del sistema (Inter / Segoe UI). Peso 600, tracking +36%, en versales.

## Activos

| Archivo | Uso |
|---------|------|
| `logo-mark.svg` | Símbolo cuadrado (app icons, favicon @ 200px). Fondo transparente. |
| `logo-full-light.svg` | Lockup horizontal con wordmark en onyx, para fondos claros. |
| `logo-full-dark.svg` | Lockup horizontal con wordmark en crema, para fondos oscuros. |
| `favicon.svg` | Variante con fondo onyx redondeado para tabs / PWA. |

## Reglas

1. **No** distorsionar el símbolo. Escalar siempre proporcionalmente.
2. **No** alterar los degradados de oro. Si necesitas monocromo, usa `#0A0E1A` plano en lugar del degradado.
3. **Espacio libre**: deja al menos el ancho de la "V" del símbolo como margen alrededor del lockup.
4. **Tamaño mínimo legible**: 24 px para el símbolo, 120 px de ancho para el lockup.
